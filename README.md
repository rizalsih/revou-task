## 📌 Tentang Project

Project ini dibuat untuk menunjukkan kemampuan membangun website interaktif menggunakan teknologi web modern tanpa framework berat.

Fokus utama:

* UI yang bersih dan modern
* Responsif di berbagai perangkat
* Interaksi yang smooth dan ringan

---

## ✨ Fitur Utama

* 🌙 Dark mode
* 🌐 Multi bahasa (Indonesia / English)
* 🎬 Animasi scroll (fade-in)
* 📂 Dropdown interaktif
* 📱 Responsive (mobile-first design)

---

## 🛠 Teknologi yang Digunakan

* HTML5
* Tailwind CSS
* JavaScript (Vanilla)

---

## 🎯 Tujuan Project

* Menampilkan kemampuan dasar frontend development
* Membangun UI tanpa ketergantungan framework besar
* Melatih pembuatan layout responsif dan interaktif

---

## 🚧 Pengembangan Selanjutnya

* Menambahkan daftar project (portfolio showcase)
* Integrasi backend untuk form kontak
* Peningkatan aksesibilitas (ARIA & keyboard navigation)
* Optimasi performa (lazy loading, minifikasi)
